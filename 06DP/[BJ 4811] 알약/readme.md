# 백준 4811번 : 알약

## Algorithm

Dynamic Programming

## Description

+ 카탈란 수(catalan number)

  + 점화식
  
    >
        C0 = 1
        
        C1 = C0 * C0
        
        C2 = C1 * C0 + C0 * C1
        
        C3 = C2 * C0 + C1 * C1 + C0 * C2
        
        C4 = C3 * C0 + C2 * C1 + C1 * C2 + C0 * C3
        
        ...
